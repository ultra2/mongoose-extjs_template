In this demo I used www.extsolutions.net to generate some parts of the source code. My theory is to automatize the most as you can, but have full control on generating process. You will agree above 20 schemas in your project.
So you need to define mongoose schemas and extjs models, requires them, validation on both sides, ect..

Generated:
- mongoose schemas (+ requires)
- extjs models (+ requires)
- ui

The partner name length must be between 5 and 10 character. It is checked both client and server side. Change this in your www.extsolutions.net project and generate again!

- create new schemas (NOTE: isCollection must be checked in if you want a new grid tab!)
